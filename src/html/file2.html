යාන්ත්‍ර විද්‍යාව දෝලන හා තරංග තාපය පදාර්ථයේ යාන්ත්‍රික ගුණ ධාරා විද්‍යුතය

දෝලන

සරල අනුවර්තීය චලිතය

\(T = \frac{2\pi}{\omega}\)

\(f = \frac{1}{T}\)

\(f = \frac{\omega}{2\pi}\)

\(a = -\omega^2{x}\)

\(v = \omega\sqrt{A^2 - x^2}\)

\(x = A\sin(\phi + \omega{t})\)

\(v = A\omega\cos(\phi + \omega t)\)

\(a = -A\omega^2\sin(\phi + \omega t)\)

\(T\) - ආවර්ත කාලය (s)
\(\omega\) - කෝණික ප්‍රවේගය (\(\mathrm{rads^{-2}}\))
\(f\) - සංඛ්‍යාතය (\(\mathrm{H_z}\))
\(a\) - ත්වරණය (\(\mathrm{ms^{-2}}\))
\(x\) - විස්ථාපනය (m)
\(A\) - විස්තාරය (m)
\(\phi\) - කලාව (rad)
\(t\) - ගතවූ කාලය (s)
\(v\) - ප්‍රවේගය (\(\mathrm{ms^{-1}}\))

තරංග

තරංගයක ප්‍රවේගය

\(V = f\lambda\)

\(V\) - තරංගයේ ප්‍රවේගය (\(\mathrm{ms^{-1}}\))
\(f\) - සංඛ්‍යාතය (\(\mathrm{H_z}\))
\(\lambda\) - තරංග ආයාමය (m)
ඇදි තන්තුවක සම්ප්‍රේෂණය වන තීර්යක් තරංගයක ප්‍රවේගය

\(V = \sqrt{\frac{T\ell}{m}}\)

\(V\) - තරංගයේ ප්‍රවේගය (\(\mathrm{ms^{-1}}\))
\(T\) - තන්තුවේ ආතතිය (N)
\(\ell\) - තන්තුවේ දිග (m)
\(m\) - තන්තුවේ ස්කන්ධය (kg)
වායුමය මාධ්‍යක සම්ප්‍රේෂණය වන අන්වායාම තරංගයක ප්‍රවේගය

\(V = \sqrt{\frac{\gamma{P}}{\rho}}\)

\(V\) - තරංගයේ ප්‍රවේගය (\(\mathrm{ms^{-1}}\))
\(\gamma\) - පරමාණුකතාව
\(P\) - වායුගෝලීය පීඩනය (Pa)
\(\rho\) - ඝණත්වය (\(\mathrm{kgm^{-3}}\))
අන්වායාම තරංග හා උෂ්ණත්වය අතර සම්බන්දය

\(V = \sqrt{\frac{\gamma{R}T}{M}}\)

\(V\) - තරංගයේ ප්‍රවේගය (\(\mathrm{ms^{-1}}\))
\(\gamma\) - පරමාණුකතාව
\(R\) - සාර්වත්‍ර වායු නියතය (\(\mathrm{Jmol^{-1}K^{-1}}\))
\(M\) - මවුලික ස්කන්ධය (\(\mathrm{kgmol^{-1}}\))
\(T\) - නිරපේක්ෂ උෂ්ණත්වය (K)
ඝන මාධ්‍යක් තුළ අන්වායාම තරංග ප්‍රවේගය

\(V = \sqrt{\frac{y}{\rho}}\)

\(V\) - තරංගයේ ප්‍රවේගය (\(\mathrm{ms^{-1}}\))
\(y\) - යංමාපාංකය (\(\mathrm{kgm^{-1}s^{-2}}\))
\(\rho\) - ඝණත්වය (\(\mathrm{kgm^{-3}}\))

ධ්වනියේ ලාක්ෂණික

ධ්වනි ත්‍රීවෘතාව

\(I = \frac{E}{At}\)

\(I\) - ධ්වනි ත්‍රීවෘතාව (\(\mathrm{Wm^{-2}}\))
\(E\) - ශක්තිය (J)
\(A\) - වර්ගඵලය (\(\mathrm{m^2}\))
\(t\) - කාලය (s)
ත්‍රීවෘතා මට්ටම

\(L = 10\lg(\frac{I}{I_o})\)

\(\Delta{L}(L_2 - L_1) = 10\lg(\frac{I_2}{I_1})\)

\(L, L_1, L_2\) - ත්‍රීවෘතා මට්ටම (dB)
\(I, I_1, I_2\) - ධ්වනි ත්‍රීවෘතාව (\(\mathrm{Wm^{-2}}\))
\(I_o\) - ශ්‍රව්‍යතා දේහලිය (\(10^{-12}\mathrm{Wm^{-2}}\))
නුගැසුම්

\(f = |f_a - f_b|\)

\(f\) - නුගැසුම් සංඛ්‍යාතය (\(\mathrm{H_z}\))
\(f_a\) - a භාන්ඩයේ සංඛ්‍යාතය (\(\mathrm{H_z}\))
\(f_b\) - b භාන්ඩයේ සංඛ්‍යාතය (\(\mathrm{H_z}\))

ඩොප්ලර් ආචරනය

\(f' = \frac{V'}{\lambda'}\)

\(f'\) - නිරික්ෂකයාට සාපේක්ෂ සංඛ්‍යාතය (\(\mathrm{H_z}\))
\(V'\) - නිරික්ෂකයාට සාපේක්ෂ ප්‍රවේගය (\(\mathrm{ms^{-1}}\))
\(\lambda'\) - නිරික්ෂකයාට සාපේක්ෂ තරංග ආයාමය (m)
ප්‍රභවය හා නිරික්ෂකයා අතර සපේක්ෂ චලිතයක් අතිවිට

\(f' = (\frac{V \pm U_o}{V \mp U_s})f\)

\(f'\) - නිරීක්ෂකයාට සාපේෂව සංඛ්‍යාතය (\(\mathrm{H_z}\))
\(f\) - සත්‍ය සංඛ්‍යාතය (\(\mathrm{H_z}\))
\(V\) - වාතයේ ධ්වනි ප්‍රවේගය (\(\mathrm{ms^{-1}}\))
\(U_s\) - ප්‍රභවයේ ප්‍රවේගය (\(\mathrm{ms^{-1}}\))
\(U_o\) - නිරීක්ෂකයාගේ ප්‍රවේගය (\(\mathrm{ms^{-1}}\))
